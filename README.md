# Code du TP sur le controle discret RST et PID.
### Installation et utilisation  
Pour utiliser les sous-programmes et les notebooks, il est recommandé d'instaler Anaconda, une distribution python qui prends en charge automaiquement les notebooks Jupyter, il sera néanmoins nécéssaire d'insaller le paquet python-control avec la commande:
'''
conda install control
'''
